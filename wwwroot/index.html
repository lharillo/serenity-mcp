Authentication

Header-Based API Key

All requests must include your Serenity Star API key in the X-Serenity-API-Key header.

VS Code Configuration (HTTP Streamable):

{
  "servers": {
    "serenity-star": {
      "type": "http",
      "url": "https://your-server.example.com/",
      "headers": {
        "X-Serenity-API-Key": "YOUR_API_KEY"
      }
    }
  }
}

Claude Desktop Configuration (via mcp-remote):

{
  "mcpServers": {
    "serenity-star": {
      "command": "npx",
      "args": [
        "-y", "mcp-remote",
        "https://your-server.example.com/",
        "--header", "X-Serenity-API-Key: YOUR_API_KEY"
      ]
    }
  }
}

Security Note: The server does not store any API keys. Your credentials are forwarded directly to the Serenity Star API with each request.
