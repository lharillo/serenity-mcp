Quick Start

Connect via SSE 

curl -N -H "Accept: text/event-stream" \
  https://mcp.starkcloud.cc/serenitystar/sse

Health Check 

curl https://mcp.starkcloud.cc/serenitystar/health

API Endpoints

GET /serenitystar/sse

Server-Sent Events endpoint for MCP protocol communication

Headers:

Accept: text/event-stream
POST /serenitystar/message

Send messages to the MCP server

Query Parameters:

sessionId=<session-id>
GET /serenitystar/health

Health check endpoint for monitoring

Response:

{"status":"healthy","timestamp":"..."}

Features

35+ Tools

Complete CRUD for agents, models, conversations, document upload, analytics, and more

Real-time SSE

Server-Sent Events for efficient, real-time communication with MCP clients

Secure & Stateless

No credentials stored server-side. API keys via headers only.

Authentication

Header-Based API Key

All requests must include your Serenity Star API key in the X-Serenity-API-Key header.

MCP Client Configuration:

{
  "mcpServers": {
    "serenity-star": {
      "url": "https://mcp.starkcloud.cc/serenitystar/sse",
      "headers": {
        "X-Serenity-API-Key": "YOUR_API_KEY"
      }
    }
  }
}

Security Note: The server does not store any API keys. Your credentials are forwarded directly to the Serenity Star API.
